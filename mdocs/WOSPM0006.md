# WOSPM0006

## Idenditiy

| Property        | Value           |
| ------------- |-------------|
| code      | WOSPM0006 |
| title      | NO_README_TOC      |
| message | Every README file should have table of contents list.     |
| type | MetricType::ERROR      |

## Rationale

A table of contents at the top of the README file will help users to see a quick short summary of the document.
